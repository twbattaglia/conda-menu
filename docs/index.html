Manage Anaconda virtual environments from your menubar

An unoffical way to launch, create, remove, clone and export all of your conda environments.
macOS (.dmg) Windows 64 bit (.exe) Windows 32 bit (.exe)
Star

Launch

Start a new Terminal or Juputer notebook session within the virtual enviornment and easily begin working.

Create

Make a new conda environment and choose different Python versions, or drag-in a Conda environment file into the window to automatically add it to your system

Remove

In a few clicks, remove any unwanted environments from your Anaconda set-up.

Share

Generate a reproducible environment file of all installed packages and dependencies from Anaconda and pip

Installation

  • macOS
  • Windows

Step 1. Install Anaconda/Miniconda

Download the graphical installer

Double-click the downloaded .pkg file and follow the instructions. Be sure to choose Anaconda as the default system Python.

Step 2. Drag to /Applications

Once Anaconda is sucessfully installed, drag Conda-menu.app into the /Applications folder and double-click to application to run.
